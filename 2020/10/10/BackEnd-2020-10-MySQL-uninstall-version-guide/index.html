MySQL 免安装版配置

为什么使用免安装版？

  • 绿色无污染
  • 便携轻量
  • 灵活可控

开始

  1. 前往 MySQL 官网下载免安装版
    MySQL Community Downloads

  1. 下载解压至任意目录，这里解压到 C:/MySQL 为例，解压后得到以下目录结构

  2. 配置环境变量：

    检查是否设置成功命令行键入 mysql --version
    如果显示失败，检查环境变量的设置

  3. 在 C:/MySQL/mysql-8.0.21-winx64/ 下新建 my.ini 文件（可选，如无特殊需求不建议自行配置）

  4. 使用管理员权限打开 CMD 执行安装命令（如果忽略 .ini 文件，一切都按默认进行配置）

  5. 执行 MySQL 初始化命令，在 mysql-8.0.21-winx64 目录下就会生成 data 文件夹
